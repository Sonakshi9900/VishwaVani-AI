## 📌 Overview

**VishwaVani AI** is a powerful web application that breaks language barriers by translating PDF documents into **15+ Indian languages**, generating AI-powered summaries, and providing text-to-speech functionality. Built with modern web technologies and powered by Google Cloud Translate API and Gemini AI.

### 🎯 Key Features

- 📄 **PDF Translation** - Translate any PDF document into 15+ Indian languages
- 🤖 **AI Summarization** - Get intelligent summaries using Gemini 2.0 AI
- 🔊 **Text-to-Speech** - Listen to translations with natural neural voices
- 💬 **AI Chatbot** - 24/7 assistant for document-related queries
- 🎯 **Preserves Formatting** - Maintains original PDF layout and structure
